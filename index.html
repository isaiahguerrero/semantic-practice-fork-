Basic text tags give meaning to your content and help the browser interpret it correctly. Many of the tags you use communicate structure to assistive technologies.

  • p - paragraph text
  • em - emphasized or stressed text
  • strong - important text
  • cite - title of a work
  • mark - highlighted or marked text
  • b - attention text without added importance
  • i - alternate voice or tone
  • u - underlined or stylistically offset text
  • kbd - keyboard input
  • samp - sample computer output
  • var - variable in code

When choosing a text tag, always think about the meaning of the content, not the styling. HTML should describe the purpose of the text, not how it looks.

Headings and Grouping Content

Headings help outline the main topics on your page. They follow a hierarchy and should always appear in order, starting with an h1 for the main topic.

  • h1 - top-level page heading
  • h2 - sub-topic under the h1
  • h3 - sub-topic under an h2
  • h4 - sub-topic under an h3
  • h5 - sub-topic under an h4
  • h6 - sub-topic under an h5
  • section - groups related content together

Think of headings the same way you would structure an outline. Each section should contain one heading that accurately describes its primary topic.

Special Characters

HTML provides entity characters for symbols that cannot be typed directly or that would otherwise break your page's code.

  • &copy; - copyright symbol
  • &nbsp; - non-breaking space
  • &amp; - ampersand
  • &lt; - less-than symbol
  • &gt; - greater-than symbol

Many entity characters are available. They can help prevent HTML from breaking and provide proper accessibility and formatting.

Using Lists

Lists help structure content that is grouped or related. They also help users with assistive technology by announcing the number of items in the list.

  • ol - ordered list for steps or counts
  • ul - unordered list for grouped items
  • li - list item used in both types of lists

You can also nest lists inside each other to organize sub-items. Lists improve readability and user experience when used appropriately.
